## Project info
🎓 Online University Exam System with AI Proctoring
📌 Project Overview

The Online University Exam System is a full-stack web application designed to conduct secure online examinations with advanced AI-based proctoring features. The platform enables universities to manage exams, monitor students in real-time, and prevent cheating using modern web technologies.

This system includes Student Panel, Admin Panel, and Proctoring System, ensuring a complete digital examination environment.

🚀 Features
👨‍🎓 Student Panel
User authentication (Login/Register)
Dashboard with:
🏠 Home
📝 Test Series
📊 Results
Attempt online exams (MCQ-based)
Timer-based test submission
View exam results and scores
Cheating Detection:
Tab switch detection
Webcam monitoring
Activity tracking
👨‍💼 Admin Panel
Create and manage exams
Add/edit/delete questions
Monitor student activity in real-time
View and analyze test scores
Access detailed reports and logs
🎥 Proctoring System
Live student video monitoring (WebRTC)
Real-time alerts:
Tab switching
Suspicious activity
Activity log tracking
Communication Features:
Live chat (Student ↔ Proctor)
Voice communication (Microphone support)

**Use your preferred IDE**

If you want to work locally using your own IDE, you can clone this repo and push changes. Pushed changes will also be reflected in Lovable.

The only requirement is having Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Follow these steps:

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and an instant preview.
npm run dev
```
